## 定義
####工作區(Working Directory)：你存檔案的地方。
####佔存區(Stage or index)：你將資料提交到.git資料夾。
####版本庫(Repository)：你將資料提交到當前分支。
## 1. 初始化
```
git init
```

## 2. 把所有Working Directory 檔案加入到Stage
```
git add .
```

## 3. 將Stage 檔案提交到 本地分支
```
git commit -m "add heroku push"
```


git branch -M main
